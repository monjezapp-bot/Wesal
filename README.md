# وصال - تطبيق مراسلة عربي 💬

تطبيق مراسلة فوري موجه للعالم العربي، مبني بـ React + Firebase

## المميزات
- ✅ تسجيل دخول برقم الهاتف
- ✅ محادثات فورية
- ✅ حالات (Stories)
- ✅ واجهة عربية كاملة RTL
- ✅ تصميم داكن احترافي

## تشغيل المشروع

```bash
npm install
npm run dev
```

## إعداد Firebase

1. اذهب إلى [console.firebase.google.com](https://console.firebase.google.com)
2. أنشئ مشروعاً جديداً
3. فعّل: Authentication (Phone) + Realtime Database + Storage
4. افتح `src/App.jsx` وضع بيانات مشروعك في `FIREBASE_CONFIG`

## النشر على Vercel

```bash
npm run build
```
ثم ارفع على [vercel.com](https://vercel.com) مجاناً
